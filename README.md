# 🏢 HRM System

A Human Resource Management (HRM) System built using **React.js, Node.js, Express.js, and MySQL**. This project aims to simplify HR operations by providing a modern and user-friendly interface for managing departments, employees, and other HR-related tasks.

> 🚧 **Project Status:** Work in Progress

## ✨ Features

- 🔐 User Authentication (Login & Signup)
- 🛡️ JWT-based Authentication
- 🏢 Department Management (CRUD Operations)
- 📋 Department Status (Active/Inactive)
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- **Frontend:** React.js, Bootstrap, CSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Authentication:** JWT, bcrypt
- **Version Control:** Git & GitHub

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/nikitabpawar20/HRM.git
```

### Install Dependencies

#### Frontend

```bash
cd client
npm install
npm run dev
```

#### Backend

```bash
cd server
npm install
npm start
```
